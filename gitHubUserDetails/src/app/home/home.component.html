Implementation Details

  • Implemented functionality to list GitHubUsers. User list is fetched using github API.
  • Visually appealing and user freindly display.
  • Registered application and use client Id & Key for API authorization.
  • On initial loading of application default result is displayed using API.
  • Implemented a feature to provide Search functionality based on User name.
  • As you type in Searchbox, service is triggered to fetch results for that user name.It is based on timeout and not for each key storke.
  • Implemented Sorting feature (Based on Username & Rank).
  • Implemented services to fetch individual user based details for repositories using github API.
  • Implemented logic on details button to show users all repositories & their associated language along with few more details.
  • Implemented logic to provide masking of screen while API interaction.
  • Implemented simple routing feature, this page is by default landing page.
  • Implemented Pagination.
